# Box-Shadow
Three.js에 익숙해지기 위해 만들어본 box 입니다.

- [Live](https://box-shadow-gamma.vercel.app)

## Getting Started

install dependencies:
```
yarn
```

Run the local server at lacalhost:3000:

```
yarn start
```

Create the Build:
```
yarn build
```

Code Test:
```
yarn test
```

## Features
- React.js
- Three.js
